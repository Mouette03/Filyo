**Self-hosted** file sharing app. Glassmorphism dark design, bilingual (FR/EN), **reverse sharing** feature.

## 🚀 Quick Start

**SQLite (recommended)**:
```bash
cp .env.example .env
docker compose up -d
```

**Access**: http://localhost:3001

## ✨ Features
- Multi-file drag & drop upload
- Password protection per link
- Expiration (1h/24h/7d/30d/never)
- Download limits per link
- **Reverse sharing** (receive files)
- Multi-user (Admin/User roles)
- Dark/Light/Auto themes
- Docker multi-arch (amd64/arm64)
